Cheet Sheet:

Display and Positioning

The CSS Display property determines the type of render block for an element

Display Descripcion
:block takes up the full width of their container with line breaks before and after
:inline take up as little space as possible
:inline-block can appear next to each other, and can have their width and height manually adjusted
Position: Descripcion
absolute Removes the element from the document flow
relative enables an element to be positioned relative to where it would have originally been on a web page
fixed it is set/pinned to a specific spot on a page
